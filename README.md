# BixinAPI

A python wrapper for `bixin-api`

## Feature

+ `BixinClient` for qr-code login
+ User info access
+ Basic data model for events/user-data
+ `Login` implementation with Redis backend 

## install 

`pip install bixin-api`

## Examples

Please view `examples/flask/app.py`
